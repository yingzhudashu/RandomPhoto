# 使用指南

## 应用内操作

### 主界面
- **下一张** — 随机加载相册中的一张照片
- **删除** — 将当前照片移到回收站
- **添加小组件** — 引导添加桌面小组件
- **回收站** — 查看已删除的照片

### 回收站
- 查看所有已删除的照片（缩略图 + 删除时间）
- **恢复** — 将照片恢复到相册（保存到 Pictures/Restored 目录）
- **永久删除** — 彻底删除，不可恢复
- **清空** — 清空所有回收站照片
- 照片在回收站保留 30 天，到期自动永久删除

## 桌面小组件

### 添加方法
1. 长按桌面空白处
2. 选择「小组件」
3. 找到「随机相册」
4. 拖动 4×3 大组件到桌面

### 按钮说明
| 按钮 | 功能 |
|------|------|
| 🎲 随机 | 随机加载一张照片 |
| 🗑️ 删除 | 移到回收站（需系统确认） |
| 🔒 隐藏 | 隐藏当前照片 |
| ♻️ 回收 | 打开回收站页面 |

### 其他操作
- 点击照片 → 随机下一张
- 30 秒无操作 → 自动隐藏照片
- 点击小组件背景 → 打开应用并同步当前照片

## 权限说明

应用需要以下权限：
- **相册读取权限** — 用于读取和展示照片
- 首次打开时会弹出权限申请
- 如果拒绝，可以在系统设置中手动开启

## 常见问题

### Q: 删除照片时提示失败？
A: Android 10+ 需要通过系统确认框才能删除非本应用创建的照片。点击删除后会弹出系统确认框，请点击「允许」。

### Q: 小组件不显示照片？
A: 请点击小组件上的「🎲 随机」按钮加载照片。首次添加时显示为空白状态。

### Q: 恢复的照片在哪里？
A: 恢复的照片保存在 `Pictures/Restored` 目录中。
