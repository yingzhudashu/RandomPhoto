# 🚀 使用教程

欢迎使用随机相册应用！本教程将帮助您快速上手。

## 📱 应用功能

随机相册是一个简洁的 Android 应用，主要功能：

1. **随机展示照片** - 从相册中随机选择一张照片展示
2. **下一张** - 点击按钮切换到另一张随机照片
3. **删除照片** - 删除当前展示的照片（需确认）

## 🎯 快速开始

### 第一步：启动应用

1. 在手机桌面找到应用图标
2. 图标为紫色背景 + 白色相机图案
3. 点击图标启动应用

### 第二步：授予权限

首次启动时，系统会弹出权限请求：

```
┌─────────────────────────────┐
│  随机相册想要访问您的照片   │
│                             │
│  此应用需要访问相册以显示   │
│  随机照片                   │
│                             │
│     [拒绝]      [允许]      │
└─────────────────────────────┘
```

**点击"允许"**以继续使用应用。

### 第三步：查看照片

授权后，应用会自动：

1. 扫描相册中的所有照片
2. 随机选择一张展示
3. 显示在主界面

## 🎮 基本操作

### 操作 1: 查看下一张照片

```
点击底部的蓝色"下一张"按钮

┌─────────────────┐
│                 │
│     [照片]      │
│                 │
│  ┌────┐ ┌────┐ │
│  │下一张│ │删除│ │ ← 点击这里
│  └────┘ └────┘ │
└─────────────────┘
```

**效果**: 随机切换到另一张照片

**提示**: 
- 每次点击都会随机选择
- 可能重复出现同一张照片
- 照片数量越多，重复概率越低

### 操作 2: 删除照片

```
1. 点击底部的红色"删除"按钮

┌─────────────────┐
│                 │
│     [照片]      │
│                 │
│  ┌────┐ ┌────┐ │
│  │下一张│ │删除│ │ ← 点击这里
│  └────┘ └────┘ │
└─────────────────┘

2. 确认删除对话框弹出

┌──────────────────────┐
│     确认删除         │
│                      │
│  确定要删除这张照片  │
│  吗？此操作不可恢复。│
│                      │
│    [取消]  [删除]    │ ← 点击这里确认
└──────────────────────┘

3. 照片被删除，自动跳转到下一张
```

**⚠️ 警告**: 
- 删除操作**不可恢复**
- 照片会从手机相册中永久删除
- 请谨慎操作！

## 💡 使用技巧

### 技巧 1: 快速浏览照片

连续点击"下一张"按钮可以快速浏览相册中的照片，适合：
- 回忆过去的照片
- 发现被遗忘的照片
- 快速筛选想删除的照片

### 技巧 2: 清理相册

使用删除功能清理不需要的照片：
1. 点击"下一张"浏览
2. 看到不想要的照片点击"删除"
3. 确认后照片被移除
4. 继续下一张...

### 技巧 3: 避免误删

删除前会弹出确认对话框：
- 如果误点删除，点击"取消"
- 确认后再点击"删除"
- 养成确认的习惯

## ❓ 常见问题

### Q: 为什么加载很慢？
**A**: 
- 首次启动需要扫描整个相册
- 照片数量多（1000+）时需要几秒
- 后续加载会更快
- 建议耐心等待首次加载完成

### Q: 照片显示不全？
**A**: 
- 照片会自适应屏幕大小
- 保持原始宽高比
- 不会裁剪或变形
- 某些超大照片可能需要更多加载时间

### Q: 删除的照片能恢复吗？
**A**: 
- **不能！** 删除是永久性的
- 照片会从相册中彻底移除
- 除非有备份，否则无法恢复
- 请谨慎使用删除功能

### Q: 应用会消耗很多流量吗？
**A**: 
- **不会**，应用完全离线运行
- 不联网，不消耗流量
- 所有操作在本地完成
- 放心使用

### Q: 应用会偷看我的照片吗？
**A**: 
- **不会**，应用只在您使用时访问照片
- 不会后台扫描
- 不会上传照片
- 不会收集任何数据
- 开源代码，透明可查

### Q: 支持视频吗？
**A**: 
- 当前版本**仅支持照片**
- 不支持视频文件
- 未来版本可能添加视频功能

### Q: 可以自定义界面吗？
**A**: 
- 当前版本界面固定
- 未来版本可能添加主题切换
- 欢迎提出建议

## 🔒 隐私说明

### 我们**不**做：
- ❌ 不收集任何个人信息
- ❌ 不上传照片到服务器
- ❌ 不使用第三方分析工具
- ❌ 不展示广告
- ❌ 不跟踪用户行为

### 我们**只**做：
- ✅ 本地读取照片用于显示
- ✅ 本地删除照片（经您确认）
- ✅ 所有数据存储在您的设备

## 🆘 遇到问题？

### 应用闪退
1. 重启应用
2. 清除应用数据（设置 → 应用 → 随机相册 → 存储 → 清除数据）
3. 重新安装

### 无法加载照片
1. 检查是否已授予相册权限
2. 确认相册中有照片
3. 重启应用

### 删除失败
1. 检查是否为云同步照片（可能无法删除）
2. 确认存储空间充足
3. 重启应用后重试

### 其他问题
1. 查看 README.md 文档
2. 检查 INSTALL.md 安装指南
3. 联系开发者获取帮助
